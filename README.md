## Description
This is an example that was inspired by Vlad Kononov's book The Study of DDD. This is an attempt to make a small piece with a multi-level architecture and Active Recording.
[decision_tree](https://user-images.githubusercontent.com/44276887/234344566-9febf11d-347f-4fe0-98be-6e6cd878e596.png)
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
